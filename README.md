# Folder Manager (Python)

A simple Python project that allows you to manage folders and text files in your system.  
With this program, you can:

- Create folders  
- Remove folders  
- Create text files  
- Update text files  
- Read text files  
- Delete text files  
- Navigate into folders  
- Go back to the previous folder  

This project is written using **Object Oriented Programming (OOP)**.
